How to use Rateify
  • Enter journal titles in the individual or bulk add space (total up to 10).
  • Ensure the correct percentile band method is selected (default: Quartiles).
  • Click Compute to generate the table (ABDC rating + Scopus metrics).
  • If ABDC shows “na” with the grey pill “Choose ABDC close match”, click it and select the closest ABDC title variant.
  • Optional: click Export CSV to download results.
Tips
  • Use the most official journal title (avoid abbreviations).
  • Matching is case-insensitive, but punctuation and symbols matter (e.g., : ; , - &).

Methodology

This note describes a method to establish ABDC equivalence for journals not listed in ABDC but listed in Scopus.

1. The ABDC journal list called the “Draft 2025 ABDC Journal Quality List” (2659 journals, listed in ABDC database) is matched with the Scopus list 2023 (29,755 journals, see Scopus database).

2. About 1702 journals are common between the two lists. For these common journals, the three scores, namely CiteScore, SNIP, and SJR, are extracted (see ABDC database).

3. By default, four quartile ranges are created for the three scores.† However, the user can change the range of the four slices, eg, for A*/A/B/C, as 10/20/20/40 or 10/30/30/30, etc.

4. In the Search tab, enter the journal name(s). Once the name(s) are entered, the search outputs the following ten values. Note that if a journal is not listed in ABDC then “na” is returned and if a journal is not listed in Scopus then “#N/A” is returned: (1) Quick code (amalgamation of #2–4–7–9–10); (2) ABDC rating; (3) CiteScore value; (4) ABDC equivalent (CiteScore); (5) Top 10% CiteScore percentile; (6) SNIP value; (7) ABDC equivalent (SNIP); (8) SJR value; (9) ABDC equivalent (SJR); (10) SJR Quartile.

The correlation between CiteScore, SNIP, and SJR for Scopus is shown in the Correlations tab.

Note: This tool is currently in test mode. Journals are matched primarily by title, with ISSN used as a secondary validation check. If ISSNs differ between sources, the result is flagged for transparency. In addition, ABDC and Scopus may use slightly different journal titles. When an exact ABDC title match is not found but a close match is likely, the tool may suggest 1–3 closest ABDC titles for user confirmation. If you notice any issues, score discrepancies, or journal title mismatches, please report them.

† The percentile bands are computed dynamically from the matched ABDC–Scopus overlap (N = 1702) using Excel-style QUARTILE.INC / PERCENTILE.INC. For journals near LL/UL boundary cutoffs, score values may fall into adjacent bands due to tied values at the threshold.
